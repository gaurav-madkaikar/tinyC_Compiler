1. A function name may be declared twice

3. The entry in the symbol table named as 'return' corresponds to the return type

Three Adress Codes (TAC)
----------------------------------------------------------------------------------------------------------------------------------

fibonacci: 
	   1:	t00 = 1
	   2:	if n >= t00 goto 5
	   3:	goto 12
	   4:	goto 14
	   5:	t01 = 1
	   6:	t02 = n - t01
	   7:	param t02
	   8:	t03 = call fibonacci, 1
	   9:	t04 = n * t03
	  10:	ret t04
	  11:	goto 
	  12:	t05 = 1
	  13:	ret t05

main: 
	  15:	t00 = 8
	  16:	n = t00
	  17:	param n
	  18:	t02 = call fibonacci, 1
	  19:	ans = t02
	  20:	param t03
	  21:	param n
	  22:	param ans
	  23:	t04 = call printf, 3
	  24:	param t05
	  25:	t06 = call printf, 1
